# Contra-Fichas
 
Projeto desenvolvido para auxiliar jogadores na criação e gerenciamento de personagens para um sistema de RPG Próprio.

A plataforma permitirá a geração aleatória de fichas, criação manual de personagens, gerenciamento de atributos, habilidades e armazenamento de fichas dos usuários.

## Tecnologias

- HTML
- CSS
- JavaScript
- Java
- Spring Boot
- MySQL

## Funcionalidades

- Criação de personagens
- Geração aleatória de fichas
- Biblioteca de classes e raças
- Salvamento de personagens
